## Overview
This project analyzes sales and customer behavior for a **Vancouver-based Canadian online scrunchie business**.
The objective is to identify revenue drivers, regional performance, and opportunities to improve profitability
within the Canadian market.

## Tools Used
- SQL
- Python (pandas, matplotlib)
- Excel
- Power BI

## Business Questions
- Which scrunchie categories generate the most revenue?
- How do discounts affect profit and order value?
- Which Canadian regions perform best?
- Are returning and VIP customers more valuable?
- Where is revenue lost due to returns?

## Dataset
All data is **synthetic** and designed to mimic real Canadian e-commerce transactions.

Files included:
- customers.csv
- orders.csv
- order_items.csv
- products.csv
- categories.csv
- regions.csv
- returns.csv

## Key Metrics
- Total & Net Revenue
- Average Order Value (AOV)
- Revenue by Region & Category
- Customer Lifetime Value (CLV)
- Return Rate

## Outcome
Insights from this analysis support better decision-making around:
- Product strategy
- Discounting
- Customer retention
- Regional growth
